# AI 模型信息展示

截至 2026 年 6 月底的国内外主要 AI 厂商与前沿模型信息展示页面。

## 在线查看

直接打开 [index.html](index.html) 即可在浏览器中查看（支持本地文件）。

或部署到 GitHub Pages / 任意静态托管服务。

## 功能

- 实时搜索模型 / 厂商
- 多选厂商筛选
- 能力过滤（推理、多模态、开放权重）
- 价格区间、上下文窗口筛选
- 表格排序
- 厂商 Logo 显示
- 详情弹窗（Modal）
- 浅色 / 深色主题切换（自动保存偏好）

## 数据来源参考

Artificial Analysis、OpenAI、Anthropic、Google DeepMind、xAI、DeepSeek、Alibaba、Zhipu、Kimi、MiniMax、Mistral 等公开资料。

**仅供演示参考**，实际价格、参数请以官方为准。

## 本地开发

无需构建工具，直接用浏览器打开 `index.html`。

## 更新

本项目为静态单页应用，修改 `data.js`、`main.js`、`styles.css` 后直接提交即可。
